Тема: termux#4


ОБЩЕДОСТУПНЫЙ САЙТ НА termux С ПОМОЩЬЮ ngrok


Всё просто!
Пишем:
apt install git
git clone https://github.com/tchelospy/termux-ngrok.git
Открываем папку с ngrok:
cd termux-ngrok
Делаем файл исполняемым:
chmod +x termux-ngrok.sh
Запускаем:
./termux-ngrok.sh
Ngrok установлен в termux, появилась новая команда:
ngrok http 8080
Если у вас есть рут то вместо 8080 можно поставить 80, если рут нету то оставьте 8080! это важно!!
Всё сайт запущен, теперь перейдя по ссылке которую нам выдал ngrok все будут видеть то что у вас в localhost
СПАСИБО ЗА ПОСЕЩЕНИЕ!
